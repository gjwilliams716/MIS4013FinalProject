Project Overview:
This project revolves around the development of a dynamic and visually engaging website dedicated to showcasing the defensive statistics of the University of Oklahoma football team. Leveraging HTML and CSS for the front-end, and PHP scripting for seamless integration with a MySQL database, the website aims to provide an intuitive and informative user experience. The primary goal of this project is to provide fans, analysts, and enthusiasts with a centralized platform to explore and analyze the defensive performance of the University of Oklahoma football team. In conclusion, we want to see where it all went wrong this year.
